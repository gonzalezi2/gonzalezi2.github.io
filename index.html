Lendit News Site

I was hired by a consultant to work for one of his projects as a front-end developer. The project was for a fintech client, Lendit, from NYC which focuses on the global financing news. The site was built on WordPress as requested by the Lendit team.

My requirements were to be in constant communication with the client and consultant as a project manager, and to relay any project developments or issues to both parties. Some of my duties with the project included editing PHP code, creating custom CSS, and using jQuery to integrate the news API that the consultant team created.

  • Client: Lendit
  • Date: August 2016 - October 2016
  • Service: Web Development
